## Getting Started

First, create your .env with values for : 
```bash
BLOB_READ_WRITE_TOKEN=
NEXTAUTH_URL=
NODE_ENV=
GOOGLE_ID=
GOOGLE_SECRET=
DISCORD_ID=
DISCORD_SECRET=
NEXTAUTH_SECRET=
RESEND_API_KEY=
```

Then, install dependencies with 
```bash
npm install
yarn add
etc...
```

Finally, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
And go to localhost:3000

Enjoy
